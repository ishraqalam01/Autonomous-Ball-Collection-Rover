A rover programmed with PBASIC that functions autonomously, with no user input.

+ Starts at a designated starting line at the start of the path
+ Rotates towards a ball
+ Opens a 3D-printed rotating arm mounted on it
+ Adjusts position for optimal pick-up
+ Closes 3D-printed rotating arm and collects ball
+ Rotates toward scoring hole and traverses through rough terrain
+ Pushes ball into scoring hole and returns to collect second ball to repeat process
